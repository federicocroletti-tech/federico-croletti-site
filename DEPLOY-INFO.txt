Backend Render environment variables required for real email delivery:
NODE_VERSION=22
CORS_ORIGINS=https://federico-croletti-site.onrender.com
CONTACT_TO_EMAIL=federico.croletti@gmail.com
CONTACT_DRY_RUN=false
SMTP_HOST=<host smtp>
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=<utente smtp>
SMTP_PASS=<password or app password smtp>
SMTP_FROM_EMAIL=<verified sender email>
SMTP_TIMEOUT_MS=10000

If SMTP is missing, the API returns CONTACT_EMAIL_NOT_CONFIGURED and the frontend shows an email fallback.
If SMTP is unreachable or misconfigured, SMTP_TIMEOUT_MS prevents the browser request from staying pending for too long.
Do not expose SMTP secrets in the frontend.
